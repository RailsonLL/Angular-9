#INTALAÇÃO DE COMPONENT EXTERNO
- baixar na raiz do projeto: ngm install --save NOVOCOMPONENTE
- adicionar no angular.json o css no Styles e o js no Scripts

#START
- criar novo modulo: ng g m NOVOMODULO
- criar novo modulo com rotas: ng g m --routing NOVOMODULO
- criar noco component: ng g c MODULO/NOVOCOMPONENTE
- criar novo service: ng g s NOVOSERVICE

- sempre que criar novo modulo adicionar no módulo o principal o import
- quando criado novo componente adiciona-lo no export do seu módulo

#Data bind
Componente > View
- interpolação: {{cliente.nome}}
- property bind: (dentro da tag html <>) bind.value="cliente.nome" ou [value]="cliente.nome"
View > Componente
- event bind: (click)="salvar(cliente)"
Two-way binding
- [(ngModel)]="cliente.nome"

#SERVICE
- os serviços são criados no módulo raiz para ficar disponível em toda a aplicação
- todo novo service deve ser declarado dentro do provider do módulo
- os services podem ser injetados em outros componentes e serviços

$REST
- para utilziar os serviços rest http é necessário o import no módulo principal do HttpClientModule

#DIRETRIZES
ngForm - seta o formulário de cadastro
ngSubmit - no formulário seta o método executado ao apertar o botão
*ngIf - verifica se deve mostrar ou não o elemento da tag
*ngFor - utilizado para impressão de arrays ou objetos na view ex: <*ngFor="let item of lista">
ngModel - faz a ligação dos atributos na view

#ROTAS
- declarar as rotas no módulo
- utilizar routerLink na view
- utilizar routes.navigate(['rota']) nos .ts
- utilizar children na configuração das rotas para organizar a página principal do componente e as sub-paginas

#DICAS
- template stream libera utilização de expreções: `` ex:`http://localhost:8080/api/clientes/${id}`
